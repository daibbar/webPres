FONCTIONS CLÉS & LOGIQUE

Tri, Recherche et Union

01

Fonction Compare (Tri)

Indispensable pour qsort. Elle trie les arêtes par poids croissant. C'est la base de l'approche gloutonne : on traite toujours l'arête la moins coûteuse d'abord.

02

Fonction Find

Recherche la racine d’un sommet en appliquant la compression de chemin, optimisant ainsi les appels futurs en réduisant la profondeur de l’arborescence.

03

Fonction Union

Fusionne deux ensembles en utilisant le rang pour attacher la racine de rang inférieur sous celle de rang supérieur, préservant l’équilibre des arbres Union-Find.

core_functions.c 

// 01. Fonction de comparaison pour qsort
int compare(const void* a, const void* b)
{
    Edge* a1 = (Edge*)a;
    Edge* b1 = (Edge*)b;
    return a1->weight - b1->weight;
}

// -----------------------------------------

// 02. Trouver le représentant (avec compression)
int find(Subset subsets[], int i)
{
    if (subsets[i].parent != i)
    {
        subsets[i].parent = find(subsets, 
                                 subsets[i].parent);
    }
    return subsets[i].parent;
}

// -----------------------------------------

// 03. Union de deux ensembles (par rang)
void unionSets(Subset subsets[], int rootX, int rootY)
{
    if (subsets[rootX].rank < subsets[rootY].rank)
        subsets[rootX].parent = rootY;
    
    else if (subsets[rootX].rank > subsets[rootY].rank)
        subsets[rootY].parent = rootX;
    
    else
    {
        subsets[rootY].parent = rootX;
        subsets[rootX].rank++;
    }
}
