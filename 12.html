05. Implémentation

FONCTION PRINCIPALE

Initialisation du graphe et appel de l'algorithme de Kruskal

1. Orchestration & Initialisation

En tant que cœur de l'implémentation, cette fonction orchestre l'ensemble du processus : elle initialise les structures...

2. Tri et Sélection

...trie les arêtes, puis itère sur ces dernières en utilisant Union-Find pour sélectionner les arêtes sûres.

3. Assemblage Final

Elle assemble progressivement l'arbre couvrant minimal tout en calculant son poids total, illustrant ainsi l'application concrète de l'algorithme.

main_kruskal.c 

void kruskal(Edge edges[], int V, int E)
{
    Edge result[V - 1]; // Arbre couvrant minimum
    Subset subsets[V];

    // Initialisation des sous-ensembles
    for (int v = 0; v < V; v++)
    {
        subsets[v].parent = v;
        subsets[v].rank = 0;
    }

    // Trier les arêtes
    qsort(edges, E, sizeof(Edge), compare);

    int e = 0; // nombre d'arêtes dans le MST
    int i = 0; // index pour les arêtes triées

    while (e < V - 1 && i < E)
    {
        Edge next = edges[i++];

        int x = find(subsets, next.src);
        int y = find(subsets, next.dest);

        // Vérifier si l'ajout crée un cycle
        if (x != y)
        {
            result[e++] = next;
            unionSets(subsets, x, y);
        }
    }
}
